# Install Guide
[Simple Video Guide](https://youtu.be/1Ug1PUPSozg)<br>

## Installing Dependencies

1. Download The Latest [MemProcFS Release](https://github.com/ufrisk/MemProcFS/releases). Download The win_x64 File. <br/>
<p align="Left">
  <img src="https://github.com/IntelSDM/RustDmaCheat/assets/86087830/3588044d-05ec-4f6e-a1e7-e1387f3a9214"
    style="width: 85%;" />
</p>
<b>Unzip it in safe location.</b><br/><br/>  
     
2. Download the [FTD3xx Dlls](https://ftdichip.com/wp-content/uploads/2024/06/FTD3XXLibrary_1.3.0.10.zip).  

<p align="Left">
  <img src="https://github.com/IntelSDM/RustDMACheat/assets/86087830/3a26c1d1-689f-48a5-897a-ca873e74b89e"
    style="width: 75%;" />
</p>
<b>Unzip it in safe location.</b><br/><br/>

Open Environment Variables:
